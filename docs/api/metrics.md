# `tabench.metrics`

The certifiers. Every scored metric — relative gaps, TSTT, system-optimum and
day-to-day measures, estimation NRMSE, the EDOC observational gaps — is
recomputed here from the flows a model emits, so a self-reported number can
never reach the leaderboard.
